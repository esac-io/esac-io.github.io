EOS

Just another org-mode based emacs configuration, or a elisp interpreter desktop distribution.

Using Emacs (elisp interpreter) as the only IO bridge between human and machine. In other words, an Emacs configuration well designed and integrated, that provides a generic computer environment for everyday use.

Shini

Shini is a lkm (linux kernel module) rootkit. Its purpose is to be simple, modular, generic (without relying on an architecture) and to segment its logical behavior from its hook methods.

GIT/SHINI

Nand2Tetris

Nand2Tetris, is a course that aims to engineer a modern computer using NAND logic gates as the elementary building block and first abstraction layer. The components to be constructed are:

  • Integrated Circuits
  • RAM/ROM
  • ALU/CPU
  • Assembler
  • Compiler
  • High-Level Language
  • Virtual Machine
  • Operational System
  • Tetris Game
